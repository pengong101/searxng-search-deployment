```
1. 主要技术突破
2. 代表性项目
3. 市场规模预测

```

3. 全球核聚变投资创历史新高

### 价值

## 🎯 使用建议

### 最佳实践

1. **明确需求** - 清晰描述调研目标
2. **细化维度** - 指定需要的信息维度
3. **多轮迭代** - 根据结果调整查询
4. **交叉验证** - 多来源信息对比

### 提示词模板

```
请调研 [主题]，包括：
1. [维度 1]
2. [维度 2]
3. [维度 3]
```

### 高级技巧

- 使用 `!bd` 仅搜索百度
- 使用 `!b` 仅搜索必应
- 使用 `!wp` 仅搜索维基百科
- 使用 `after:2025` 限制时间范围
